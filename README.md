# Time Series Forecasting

##  Project Overview
This project focuses on forecasting COVID-19 deaths in Australia using advanced time series techniques.

##  Methods Used
- Holt’s Linear Exponential Smoothing
- Damped Trend Model
- ARIMA
- Prophet
- Bootstrapped ARIMA

##  Key Results
- Prophet and Bagged ARIMA achieved the best performance.
- Smoothing models showed higher errors.
- The data showed strong trends with no stable seasonality.
